# JS .pop() function

You can remove data from the end of an array with `.pop()`.

```javascript
const myArray = [["John", 23], ["cat", 2]];
const removedFromMyArray = myArray.pop();
```

`myArray` is now equal to `["John", 23]` and `removedFromMyArray` is now equal to `["cat", 2]`.
