[[JavaScript]]
# Data Types

There are eight different data types:
1. [[Undefined]]
2. [[Null]]
3. [[Boolean]]
4. [[String]]
5. [[Symbol]]
6. [[Number]]
7. [[Object]]
8. [[BigInt]]

They can be used to assign a value to [[JS Variables]].

Strings and numbers are the most common ones.
Strings are any text.
Numbers are numbers.
