# JS .push() function

You can have pinned data to the end of an array with `.push()`.

```javascript
const myArray = [["John", 23], ["cat", 2]];
myArray.push(["dog", 3]);
```

`myArray.push(["dog", 3])` appends the value stored in the function to the `myArray` variable.
